–í—Å–µ —Ñ—É–Ω–∫—Ü–∏–∏ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç –∑–Ω–∞—á–µ–Ω–∏–µ. –ï—Å–ª–∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ–º–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –Ω–µ —É–∫–∞–∑–∞–Ω–æ, —Ç–æ –æ–ø–µ—Ä–∞—Ç–æ—Ä `return null;` –Ω–µ—è–≤–Ω–æ –¥–æ–±–∞–≤–ª—è–µ—Ç—Å—è –∫ —Ç–µ–ª—É —Ñ—É–Ω–∫—Ü–∏–∏.

## –û–ø–µ—Ä–∞—Ç–æ—Ä—ã

Dart –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã, —É–∫–∞–∑–∞–Ω–Ω—ã–µ –≤ —Å–ª–µ–¥—É—é—â–µ–π —Ç–∞–±–ª–∏—Ü–µ. –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –º–æ–∂–Ω–æ  —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –∫–∞–∫ —á–ª–µ–Ω—ã –∫–ª–∞—Å—Å–∞.

> **–ü—Ä–µ–¥—É–ø—Ä–µ–∂–¥–µ–Ω–∏–µ**: –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ - —ç—Ç–æ –ø—Ä–∏–±–ª–∏–∑–∏—Ç–µ–ª—å–Ω–æ–µ –ø–æ–≤–µ–¥–µ–Ω–∏–µ –ø–∞—Ä—Å–µ—Ä–∞ Dart. –ß—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å —Ç–æ—á–Ω—ã–µ –æ—Ç–≤–µ—Ç—ã, –æ–±—Ä–∞—Ç–∏—Ç–µ—Å—å –∫ –≥—Ä–∞–º–º–∞—Ç–∏–∫–µ –≤ [—Å–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ü–∏–∏ —è–∑—ã–∫–∞ Dart](https://dart.dev/guides/language/spec).

–ò—Å–ø–æ–ª—å–∑—É—è –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã, —Å–æ–∑–¥–∞—é—Ç—Å—è –≤—ã—Ä–∞–∂–µ–Ω–∏—è. –í–æ—Ç –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø—Ä–∏–º–µ—Ä–æ–≤ –æ–ø–µ—Ä–∞—Ç–æ—Ä–Ω—ã—Ö –≤—ã—Ä–∞–∂–µ–Ω–∏–π:

–í —Ç–∞–±–ª–∏—Ü–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ –∫–∞–∂–¥—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä –∏–º–µ–µ—Ç –±–æ–ª–µ–µ –≤—ã—Å–æ–∫–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç, —á–µ–º –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã –≤ —Å–ª–µ–¥—É—é—â–∏—Ö –∑–∞ –Ω–∏–º —Å—Ç—Ä–æ–∫–∞—Ö. –ù–∞–ø—Ä–∏–º–µ—Ä, –º—É–ª—å—Ç–∏–ø–ª–∏–∫–∞—Ç–∏–≤–Ω—ã–π –æ–ø–µ—Ä–∞—Ç–æ—Ä `%` –∏–º–µ–µ—Ç –±–æ–ª–µ–µ –≤—ã—Å–æ–∫–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç, —á–µ–º (–∏, —Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ, –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è —Ä–∞–Ω—å—à–µ) –æ–ø–µ—Ä–∞—Ç–æ—Ä —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ `==`, –∫–æ—Ç–æ—Ä—ã–π –∏–º–µ–µ—Ç –±–æ–ª–µ–µ –≤—ã—Å–æ–∫–∏–π –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç, —á–µ–º –ª–æ–≥–∏—á–µ—Å–∫–∏–π –æ–ø–µ—Ä–∞—Ç–æ—Ä –ò `&&`. –≠—Ç–æ—Ç –ø—Ä–∏–æ—Ä–∏—Ç–µ—Ç –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ —Å–ª–µ–¥—É—é—â–∏–µ –¥–≤–µ —Å—Ç—Ä–æ–∫–∏ –∫–æ–¥–∞ –≤—ã–ø–æ–ª–Ω—è—é—Ç—Å—è –æ–¥–∏–Ω–∞–∫–æ–≤–æ:

```dart

// Harder to read, but equivalent.

```

> **–ü—Ä–µ–¥—É–ø—Ä–µ–∂–¥–µ–Ω–∏–µ**: –¥–ª—è –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –ø—Ä–∏–Ω–∏–º–∞—é—Ç –¥–≤–∞ –æ–ø–µ—Ä–∞–Ω–¥–∞, –∫—Ä–∞–π–Ω–∏–π –ª–µ–≤—ã–π –æ–ø–µ—Ä–∞–Ω–¥ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç, –∫–∞–∫–æ–π –º–µ—Ç–æ–¥ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è. –ù–∞–ø—Ä–∏–º–µ—Ä, –µ—Å–ª–∏ —É –≤–∞—Å –µ—Å—Ç—å –æ–±—ä–µ–∫—Ç `Vector` –∏ –æ–±—ä–µ–∫—Ç `Point`, —Ç–æ `aVector + aPoint` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Å–ª–æ–∂–µ–Ω–∏–µ `Vector` `+`.

### –ê—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã

Dart –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –æ–±—ã—á–Ω—ã–µ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã, –∫–∞–∫ –ø–æ–∫–∞–∑–∞–Ω–æ –≤ —Å–ª–µ–¥—É—é—â–µ–π —Ç–∞–±–ª–∏—Ü–µ.

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ó–Ω–∞—á–µ–Ω–∏–µ                                                     |
| -------- | ------------------------------------------------------------ |
| `+`      | –°–ª–æ–∂–µ–Ω–∏–µ                                                     |
| `-`      | –í—ã—á–∏—Ç–∞–Ω–∏–µ                                                    |
| `-expr`  | –£–Ω–∞—Ä–Ω—ã–π –º–∏–Ω—É—Å, —Ç–∞–∫–∂–µ –∏–∑–≤–µ—Å—Ç–Ω—ã–π –∫–∞–∫ –æ—Ç—Ä–∏—Ü–∞–Ω–∏–µ (–ø–æ–º–µ–Ω—è—Ç—å –∑–Ω–∞–∫ –≤—ã—Ä–∞–∂–µ–Ω–∏—è) |
| `*`      | –£–º–Ω–æ–∂–µ–Ω–∏–µ                                                    |
| `/`      | –î–µ–ª–µ–Ω–∏–µ                                                      |
| `~/`     | –î–µ–ª–µ–Ω–∏–µ –Ω–∞—Ü–µ–ª–æ                                               |
| `%`      | –û—Å—Ç–∞—Ç–æ–∫ –æ—Ç –¥–µ–ª–µ–Ω–∏—è –Ω–∞—Ü–µ–ª–æ                                    |

–ü—Ä–∏–º–µ—Ä—ã:

```dart

```

Dart —Ç–∞–∫–∂–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç –∫–∞–∫ –ø—Ä–µ—Ñ–∏–∫—Å–Ω—ã–µ, —Ç–∞–∫ –∏ –ø–æ—Å—Ç—Ñ–∏–∫—Å–Ω—ã–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã —É–≤–µ–ª–∏—á–µ–Ω–∏—è –∏ —É–º–µ–Ω—å—à–µ–Ω–∏—è.

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ó–Ω–∞—á–µ–Ω–∏–µ                                       |
| -------- | ---------------------------------------------- |
| `++var`  | `var = var + 1` (–∑–Ω–∞—á–µ–Ω–∏–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è `var + 1`) |
| `var++`  | `var = var + 1` (–∑–Ω–∞—á–µ–Ω–∏–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è `var`)     |
| `--var`  | `var = var - 1` (–∑–Ω–∞—á–µ–Ω–∏–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è `var - 1`) |

–ü—Ä–∏–º–µ—Ä—ã:

```dart

```

### –û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ –∏ –æ—Ç–Ω–æ—à–µ–Ω–∏—è

–í —Å–ª–µ–¥—É—é—â–µ–π —Ç–∞–±–ª–∏—Ü–µ –ø–µ—Ä–µ—á–∏—Å–ª–µ–Ω—ã –∑–Ω–∞—á–µ–Ω–∏—è –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–≤ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ –∏ –æ—Ç–Ω–æ—à–µ–Ω–∏—è.

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ó–Ω–∞—á–µ–Ω–∏–µ         |
| -------- | ---------------- |
| `==`     | —Ä–∞–≤–Ω–æ            |
| `!=`     | –Ω–µ —Ä–∞–≤–Ω–æ         |

| `<`      | –º–µ–Ω—å—à–µ           |
| `>=`     | –±–æ–ª—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ |
| `<=`     | –º–µ–Ω—å—à–µ –∏–ª–∏ —Ä–∞–≤–Ω–æ |

–ß—Ç–æ–±—ã –ø—Ä–æ–≤–µ—Ä–∏—Ç—å, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç –ª–∏ –¥–≤–∞ –æ–±—ä–µ–∫—Ç–∞ `x` –∏ `y` –æ–¥–Ω–æ –∏ —Ç–æ –∂–µ, –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä `==`. (–í —Ä–µ–¥–∫–æ–º —Å–ª—É—á–∞–µ, –∫–æ–≥–¥–∞ –≤–∞–º –Ω—É–∂–Ω–æ –∑–Ω–∞—Ç—å, —è–≤–ª—è—é—Ç—Å—è –ª–∏ –¥–≤–∞ –æ–±—ä–µ–∫—Ç–∞ –æ–¥–Ω–∏–º –∏ —Ç–µ–º –∂–µ –æ–±—ä–µ–∫—Ç–æ–º, –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ –≤–º–µ—Å—Ç–æ —ç—Ç–æ–≥–æ —Ñ—É–Ω–∫—Ü–∏—é [identify ()](https://api.dart.dev/stable/dart-core/identical.html)) –í–æ—Ç –∫–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç –æ–ø–µ—Ä–∞—Ç–æ—Ä `==`:

1. –ï—Å–ª–∏ `x` –∏–ª–∏ `y` —Ä–∞–≤–Ω—ã `null`, –≤–µ—Ä–Ω—É—Ç—å –∏—Å—Ç–∏–Ω–∞, –µ—Å–ª–∏ –æ–±–∞ —Ä–∞–≤–Ω—ã `null`, –∏ –ª–æ–∂—å, –µ—Å–ª–∏ —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω –∏–∑ –Ω–∏—Ö —Ä–∞–≤–µ–Ω `null`.
2. –í–µ—Ä–Ω—É—Ç—å —Ä–µ–∑—É–ª—å—Ç–∞—Ç –≤—ã–∑–æ–≤–∞ –º–µ—Ç–æ–¥–∞ `x. == (y)`. (–¢–∞–∫–∏–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä—ã, –∫–∞–∫ `==`, —è–≤–ª—è—é—Ç—Å—è –º–µ—Ç–æ–¥–∞–º–∏, –∫–æ—Ç–æ—Ä—ã–µ –≤—ã–∑—ã–≤–∞—é—Ç—Å—è –¥–ª—è –∏—Ö –ø–µ—Ä–≤–æ–≥–æ –æ–ø–µ—Ä–∞–Ω–¥–∞. )

### –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –ø—Ä–æ–≤–µ—Ä–∫–∏ —Ç–∏–ø–∞

–û–ø–µ—Ä–∞—Ç–æ—Ä—ã `as`, `is` –∏ `is!` —É–¥–æ–±–Ω—ã –¥–ª—è –ø—Ä–æ–≤–µ—Ä–∫–∏ —Ç–∏–ø–æ–≤ –≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è.

| –û–ø–µ—Ä–∞—Ç–æ—Ä | –ó–Ω–∞—á–µ–Ω–∏–µ                                                     |
| -------- | ------------------------------------------------------------ |
| `as`     | –ü—Ä–∏–≤–µ–¥–µ–Ω–∏–µ —Ç–∏–ø–æ–≤ (—Ç–∞–∫–∂–µ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —É–∫–∞–∑–∞–Ω–∏—è –ø—Ä–µ—Ñ–∏–∫—Å–æ–≤ –±–∏–±–ª–∏–æ—Ç–µ–∫–∏) |
| `is`     | –ò—Å—Ç–∏–Ω–∞, –µ—Å–ª–∏ –æ–±—ä–µ–∫—Ç –∏–º–µ–µ—Ç —É–∫–∞–∑–∞–Ω–Ω—ã–π —Ç–∏–ø                      |

–†–µ–∑—É–ª—å—Ç–∞—Ç `obj is T` –∏—Å—Ç–∏–Ω–∞, –µ—Å–ª–∏ `obj` —Ä–µ–∞–ª–∏–∑—É–µ—Ç –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å, —É–∫–∞–∑–∞–Ω–Ω—ã–π –≤ `T`. –ù–∞–ø—Ä–∏–º–µ—Ä, `obj is Object` –≤—Å–µ–≥–¥–∞ –∏—Å—Ç–∏–Ω–Ω–æ.

–ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ –æ–ø–µ—Ä–∞—Ç–æ—Ä `as` –¥–ª—è –ø—Ä–∏–≤–µ–¥–µ–Ω–∏—è –æ–±—ä–µ–∫—Ç–∞ –∫ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–º—É —Ç–∏–ø—É —Ç–æ–≥–¥–∞ –∏ —Ç–æ–ª—å–∫–æ —Ç–æ–≥–¥–∞, –∫–æ–≥–¥–∞ –≤—ã —É–≤–µ—Ä–µ–Ω—ã, —á—Ç–æ –æ–±—ä–µ–∫—Ç –æ—Ç–Ω–æ—Å–∏—Ç—Å—è –∫ —ç—Ç–æ–º—É —Ç–∏–ø—É. –ü—Ä–∏–º–µ—Ä:


–ï—Å–ª–∏ –≤—ã –Ω–µ —É–≤–µ—Ä–µ–Ω—ã, —á—Ç–æ –æ–±—ä–µ–∫—Ç –æ—Ç–Ω–æ—Å–∏—Ç—Å—è –∫ —Ç–∏–ø—É `T`, –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ `is T` –¥–ª—è –ø—Ä–æ–≤–µ—Ä–∫–∏ —Ç–∏–ø–∞ –ø–µ—Ä–µ–¥ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –æ–±—ä–µ–∫—Ç–∞.

```dart
  // Type check
  emp.firstName = 'Bob';

```

> **–ü—Ä–∏–º–µ—á–∞–Ω–∏–µ**: –ö–æ–¥ –Ω–µ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–µ–Ω. –ï—Å–ª–∏ `emp` –∏–º–µ–µ—Ç –∑–Ω–∞—á–µ–Ω–∏–µ `null ` –∏–ª–∏ –Ω–µ —è–≤–ª—è–µ—Ç—Å—è `Person`, —Ç–æ –≤ –ø–µ—Ä–≤–æ–º –ø—Ä–∏–º–µ—Ä–µ –≤—ã–¥–∞–µ—Ç—Å—è –∏—Å–∫–ª—é—á–µ–Ω–∏–µ; –≤—Ç–æ—Ä–æ–π –Ω–∏—á–µ–≥–æ –Ω–µ —Å–¥–µ–ª–∞–µ—Ç.

### –û–ø–µ—Ä–∞—Ç–æ—Ä—ã –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è

–ó–Ω–∞—á–µ–Ω–∏—è –º–æ–∂–Ω–æ –ø—Ä–∏—Å–≤–∞–∏–≤–∞—Ç—å —Å –ø–æ–º–æ—â—å—é –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞ `=`.  –ß—Ç–æ–±—ã –ø—Ä–∏—Å–≤–æ–∏—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π —Ç–æ–ª—å–∫–æ –µ—Å–ª–∏ –æ–Ω–∞ —Ä–∞–≤–Ω–æ–∞ `null` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –æ–ø–µ—Ä–∞—Ç–æ—Ä `??=`.

```dart

// Assign value to b if b is null; otherwise, b stays the same
```
